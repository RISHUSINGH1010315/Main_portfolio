# Rishu Singh — Developer Portfolio

A premium, high-performance, and responsive developer portfolio built to showcase engineering projects, technical expertise, and professional experience.

Live Site: [https://rishu-singh-portfolio-2026.web.app](https://rishu-singh-portfolio-2026.web.app)

---

## 🌟 Key Features

- **Premium Glassmorphic Design**: A visually stunning dark-themed design system leveraging modern transparency, gradients, and micro-interactions.
- **Fluid Responsiveness**: Designed using modern layout principles to scale seamlessly across mobile, tablet, and desktop devices.
- **Dynamic Filter System**: An interactive project showcase allowing users to filter works by categories (Web Dev, AI & ML, Creative UI).
- **Typing Hero Animation**: An engaging client-side typing animation highlighting core capabilities in real time.
- **Professional Odyssey**: An interactive timeline showcasing professional experiences, internships, and key accomplishments.
- **Firebase Deployment**: Configured for production-ready, lightning-fast hosting and CDN delivery.

---

## 🛠️ Technical Stack

- **Structure & Layout**: HTML5, Semantic DOM architecture.
- **Styling**: Tailwind CSS (fully responsive layout and custom glassmorphism utilities).
- **Logic & Interactions**: Vanilla JavaScript (ES6+) for light-weight state manipulation, filtering, and animation effects.
- **Typography & Assets**: Outfit & Inter google web fonts, Google Material Symbols.
- **Deployment**: Firebase Hosting & CDN.

## 🚀 Getting Started & Local Development

### 1. View Locally
To run the portfolio on your local system, you can open the `index.html` file directly in any modern browser, or spin up a local development server:
```bash
# Using Python
python -m http.server 8000

# Using Node.js (with live-server or serve)
npx serve .
```

### 2. Deploy to Firebase
Make sure you have `firebase-tools` installed globally or run via `npx`:
```bash
# Log in to Firebase
npx firebase-tools login

# Deploy to Firebase Hosting
npx firebase-tools deploy
```
